ad is a theme for [mdcss].

## Usage

Add [mdcss] and [ad] to your build tool:

```bash
npm install mdcss --save-dev
npm install mdcss-theme-ad --save-dev
```

Whenever [mdcss] is used, reference this theme.

```js
require('mdcss')({
	theme: require('mdcss-theme-ad')({ /* options */ })
})
```
